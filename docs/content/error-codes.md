
# 에러코드

## Error Codes

### 🔐 인증 (Authentication)

| Code | HTTP Status | Description |
|------|-------------|-------------|
| 1000 | 401 | 인증이 올바르지 않습니다 |
| 1001 | 403 | 권한이 부족합니다 |

---

### 📦 주문 (Orders)

| Code | HTTP Status | Description |
|------|-------------|-------------|
| 2000 | 400 | 주문 요청이 올바르지 않습니다 |
| 2001 | 400 | 현재 주문이 마감된 상태입니다. 다음 주문 가능 시간에 다시 시도해 주세요. |
| 2002 | 400 | 요청한 상품이 모두 주문 불가 상태입니다. |
| 2003 | 400 | 현재 주문이 불가한 지사입니다. 담당자에게 문의해 주세요. |
| 2004 | 400 | 이미 진행 중인 주문이 있습니다. |
| 2005 | 400 | 업체의 배송지 주소 정보가 없습니다. 담당자에게 문의해 주세요. |

---

### 🍅 상품 (Products)

| Code | HTTP Status | Description |
|------|-------------|-------------|
| 3000 | 400 | 상품 요청이 올바르지 않습니다 |

---

### ⚙️ 공통 (Common)

| Code | HTTP Status | Description |
|------|-------------|-------------|
| 9000 | 400 | 요청이 올바르지 않습니다 |
| 9001 | 404 | 정보를 찾을 수 없습니다 |
| 9002 | 405 | 허용되지 않은 메서드입니다 |
| 9003 | 415 | 지원하지 않는 Content-Type 입니다 |
| 9999 | 500 | 서버 오류가 발생했습니다 |
